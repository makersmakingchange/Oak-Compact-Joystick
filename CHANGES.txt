------OAK COMPACT JOYSTICK------
           Changelog
===============================
Version 1.0.1 2024-Oct-10
 - Added CAD files for Topper_Nut, Topper_Collet, and Topper_Interface

Version 1.0 2023-Sept-27
 - Updated enclosure cable tie anchor and internal labelling
 - Updated topper sizes
 - Updated documentation 
    - Included a topper guide
    - Consolidated maker documents into a single Maker Guide
 - No longer paried specifically with the Enabled Controller Mini


Version 0.9.1 2023-July-24
 - Ability to switch operation mode through button push on reset.
 - Operating mode indicated by color

Version 0.9    2023-Jun-30
- Enclosure optimised for reduced print time and improved build redundancy
- Topper connection collet design
- Updated toppers
- 5 parametric toppers
- Updated documentation to be for just the analog variant build
- Paired the analog build with Enabled Controller Mini and custom Mouse and Gamepad code

Version 0.2 	2023-May-18
 - Initial commit
 - Oak Compact Joystick with 4 optional toppers and an optional camera mount adapter
 - Both USB and analog variants available
